# 💸 Smart Expense Analyzer

A full-stack expense tracking and analytics web application built using **Django (Backend)** and **React (Frontend)**. This app allows users to track expenses, visualize spending patterns, and manage their financial data efficiently.

---

## 🚀 Features

* 🔐 User Authentication (JWT based login)
* 📊 Interactive Dashboard with Pie Chart
* 🔍 Category-based Expense Filtering
* ➕ Add Expense functionality
* ❌ Delete Expense functionality
* 🎨 Animated UI with Framer Motion
* 📈 Real-time Expense Visualization
* 🧠 Clean and scalable architecture

---

## 🛠️ Tech Stack

### Backend:

* Django
* Django REST Framework
* Simple JWT Authentication

### Frontend:

* React (Vite)
* Recharts (for charts)
* Framer Motion (for animations)
* Axios (API calls)

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Hiteshwar502133/expense-analyzer.git
cd expense-analyzer
```

---

### 2. Backend Setup (Django)

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```

---

### 3. Frontend Setup (React)

```bash
cd expense-frontend
npm install
npm run dev
```

---

## 🌐 API Endpoints

* `POST /api/login/` → Login
* `GET /api/expenses/` → Get expenses
* `POST /api/expenses/` → Add expense
* `DELETE /api/expenses/:id/` → Delete expense
